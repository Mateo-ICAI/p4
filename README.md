# Animales Chulos Chulos

Este repositorio contiene el código correspondiente a la **Práctica 4: Página web estática**.

## Objetivo del Proyecto

El objetivo de esta práctica es crear una web completa y estática sobre un carrito de compras (en este caso, de una tienda de animales llamada "Animales chulos chulos"), empleando conocimientos básicos de HTML y CSS. 

La web consta de una página principal y tres páginas interiores interconectadas entre sí a través de un menú de navegación.

## Estructura del Proyecto

El proyecto está compuesto por los siguientes archivos:

* `index.html`: Página principal.
* `catalogo.html`: Muestra el catálogo de animales disponibles (León, Piraña y Mosquito) junto con sus imágenes, descripciones y precios.
* `carrito.html`: Página que simula el resumen de la compra con los artículos seleccionados y el precio total.
* `contacto.html`: Página con un formulario para que los usuarios puedan contactar o hacer peticiones.
* `style.css`: Archivo de hojas de estilo en cascada que da un diseño uniforme, personal y atractivo a todas las páginas HTML.
* `leon.jpg`, `mosquito.jpg`, `pirana.jpg`: Imagenes utilizados en el catálogo.

## Requisitos de la Práctica Cumplidos

Tal y como se pide en el enunciado, el proyecto implementa de manera exitosa los siguientes elementos HTML:

* **Títulos:** Uso de etiquetas headers `<h1>` y `<h2>` para estructurar la información en todas las páginas.
* **Párrafos:** Textos descriptivos utilizando la etiqueta `<p>`.
* **Tablas:** Empleadas tanto en `catalogo.html` (para listar los productos) como en `carrito.html` (para el resumen de la compra).
* **Multimedia:** Imágenes integradas correctamente en el catálogo mediante la etiqueta `<img>`.
* **Formularios Web:** Un formulario completo en `contacto.html` que incluye campos de texto libre (`<input type="text">`, `<textarea>`), correo electrónico (`<input type="email">`), y listas desplegables (`<select>`).
* **Estilos:** Un diseño unificado a través del archivo `style.css`.
